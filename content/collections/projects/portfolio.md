## es

Un portfolio bilingüe de alto rendimiento que permite la gestión total de contenidos mediante un sistema híbrido de archivos Markdown y Sanity CMS. Utiliza el sistema de diseño propio (Core UI) alojado en GitHub Packages para garantizar una interfaz coherente y profesional.

## en

A high-performance bilingual portfolio allowing full content management through a hybrid system of Markdown files and Sanity CMS. It utilizes the custom design system (Core UI) hosted on GitHub Packages to ensure a consistent and professional interface.
