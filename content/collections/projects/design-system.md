## es

Core UI es una librería de componentes base y sistema de diseño privado construido para unificar y escalar el desarrollo de productos digitales. La arquitectura se centra en entregar bloques de construcción altamente personalizables con estricto cumplimiento de los estándares de accesibilidad web (A11y), garantizando consistencia visual y de comportamiento en todo el ecosistema.

## en

Core UI is a foundation component library and private design system built to unify and scale digital product development. The architecture focuses on delivering highly customizable building blocks with strict adherence to web accessibility standards (A11y), ensuring visual and behavioral consistency across the ecosystem.
